# Select

For making one selection from a large list of options.

If you need to fetch options for the select as the interacts with the select, please use the [AsyncSelect](/async-select).

<Playground {...config} />

## Variations

### MultiSelect

Give users the ability to select multiple options with the multiselect. Note that the value will now be represented as an Array rather than a string.

<Example componentName="select" scope={{ Select }} example="Multi" />

### Replacing Inner Components

Using the `components` prop you can augment the inner components of the select. See the following example of adding a component to each option in the dropdown:

<Example componentName="select" scope={{ Select, SelectOption, Box }} example="CustomOptions" showEditor />

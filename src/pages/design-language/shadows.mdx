# Shadows

There are three shadows available to provide depth to an interface.

## Usage

Shadows can be added to the Box component using the `boxShadow` prop. Focus and Error states come built-in to inputs already, and all of our shadow values are stored in the [theme](/theme) for referencing anywhere else.
